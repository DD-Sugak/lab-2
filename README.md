##Описание
Программа для поиска и проверки математических выражений в тексте и файлах. Реализована на Python с использованием регулярных выражений.

##Основные функции
Проверка корректности математических выражений

Поиск выражений в тексте и файлах

Поддержка операторов: +, -, *, /, %

Работа с дробными числами и скобками

Полное покрытие unit-тестами
